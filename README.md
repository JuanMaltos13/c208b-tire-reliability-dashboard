# c208b-tire-reliability-dashboard
Power BI dashboard project for analyzing tire replacement reliability trends on a fleet of Cessna 208B Grand Caravan aircraft.

# C208B Tire Reliability Dashboard
- Rolling averages
- Tire distribution analysis
- Fleet utilization ranking

---

# Key KPIs

- Total Tire Replacements
- Tire Replacement Rate
- Flight Hours
- Average Replacements per Aircraft
- Main Gear Replacement Ratio
- Nose Gear Replacement Ratio

---

# Technologies Used

- Microsoft Power BI
- Microsoft Excel
- CSV Data Sources
- DAX
